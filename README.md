## **Installation**

**Using** `poetry`

```
git clone https://github.com/TETRIX8/snapinsta.git
cd snapinsta
poetry shell
poetry install
```

**Using** `pip`

```
git clone https://github.com/TETRIX8/snapinsta.git
cd snapinsta
virtualenv env
env/scripts/activate
pip install -r requirements.txt
```
